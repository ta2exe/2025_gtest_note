数学基礎

AIと機械学習の理解には、確率統計と微分積分の基礎知識が不可欠です。これらの数学的概念は、アルゴリズムの動作原理を理解し、モデルの評価・改善を行う上での基盤となります。

最適化数学

微分・積分

機械学習における最適化では、損失関数を最小化するために勾配降下法が用いられます。

$$\frac{\partial}{\partial \theta} J(\theta) = \frac{\partial}{\partial \theta} \frac{1}{2m} \sum_{i=1}^{m} (h_\theta(x^{(i)}) - y^{(i)})^2$$

勾配降下法

🔄 最適化アルゴリズム

$$\theta_{t+1} = \theta_t - \alpha \nabla_\theta J(\theta_t)$$ $$\text{where } \alpha \text{: learning rate}$$

最小二乗法

線形回帰の基本原理である最小二乗法は、誤差の二乗和を最小化します。

$$\min_{\theta} \sum_{i=1}^{n} (y_i - \theta^T x_i)^2$$

統計学基礎

確率理論

確率分布

  • 確率変数：実験の結果を数値で表現する変数
  • 確率密度関数：連続確率変数の分布を記述する関数
  • 確率質量関数：離散確率変数の分布を記述する関数

条件付き確率

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

💡 ベイズの定理

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

主要確率分布

正規分布

$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

ベルヌーイ分布

$$P(X = k) = p^k(1-p)^{1-k}, \quad k \in \{0, 1\}$$

二項分布

$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

ポアソン分布

$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$$

記述統計

中心傾向

平均（期待値）

$$E[X] = \bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$$

中央値

データを大小順に並べたときの中央の値。外れ値の影響を受けにくい。

最頻値

データ中で最も頻繁に現れる値。カテゴリデータの代表値として使用。

ばらつきの指標

分散・標準偏差

$$\text{Var}(X) = \sigma^2 = E[(X - \mu)^2] = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2$$ $$\text{標準偏差} = \sigma = \sqrt{\text{Var}(X)}$$

共分散

$$\text{Cov}(X,Y) = E[(X - E[X])(Y - E[Y])]$$

相関係数

$$r = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$$

⚠️ 疑似相関

実際には因果関係がないにもかかわらず、第三の変数の影響により見かけ上相関があるように見える現象。データ分析時の注意点。

偏相関係数

第三の変数の影響を除いた時の2変数間の相関の強さ。疑似相関を排除するために使用。

距離と類似度

距離尺度

ユークリッド距離

$$d(x, y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)^2}$$

マハラノビス距離

$$d_M(x, y) = \sqrt{(x - y)^T S^{-1} (x - y)}$$ $$\text{where } S \text{: covariance matrix}$$

コサイン類似度

$$\cos(\theta) = \frac{A \cdot B}{|A| |B|} = \frac{\sum_{i=1}^{n} A_i B_i}{\sqrt{\sum_{i=1}^{n} A_i^2} \sqrt{\sum_{i=1}^{n} B_i^2}}$$

統計的推定と検定

パラメータ推定

最尤法

観測されたデータが最も尤もらしく生成されるパラメータを推定する方法。

$$\hat{\theta}_{ML} = \arg\max_\theta L(\theta) = \arg\max_\theta \prod_{i=1}^{n} f(x_i | \theta)$$

仮説検定

🎯 仮説検定の構造

  • 帰無仮説（H₀）：効果がないという仮説
  • 対立仮説（H₁）：効果があるという仮説
  • p値：帰無仮説が正しいと仮定した時の観測結果の確率
  • 有意水準（α）：第一種の誤りを犯す確率の上限

情報理論

相互情報量

2つの確率変数間の依存関係の強さを表す指標。特徴選択で重要。

$$I(X;Y) = \sum_{x,y} p(x,y) \log \frac{p(x,y)}{p(x)p(y)}$$

エントロピー

$$H(X) = -\sum_{x} p(x) \log p(x)$$

時系列解析

移動平均

時系列データの平滑化手法。トレンドの把握やノイズ除去に使用。

$$MA_t = \frac{1}{n}\sum_{i=0}^{n-1} x_{t-i}$$

度数分布

データの各値や階級の出現頻度をまとめた表やグラフ。データの分布形状を把握するための基本的手法。

外れ値の検出

📊 外れ値検出手法

  • IQR法：四分位範囲の1.5倍を超える値
  • Z-score法：標準化後の絶対値が3を超える値
  • 統計的検定：Grubbs検定、Dixon検定等
  • 機械学習手法：Isolation Forest、Local Outlier Factor等

試験対策のポイント

🎯 G検定で頻出する内容

  • 正規分布の性質：68-95-99.7ルール、標準化の意味
  • 相関と因果：相関係数の解釈、疑似相関の概念
  • 確率分布の特徴：ベルヌーイ、二項、ポアソン分布の適用場面
  • 距離尺度の使い分け：ユークリッド距離とマハラノビス距離
  • 最小二乗法：線形回帰との関係、最適化の意味
  • ベイズの定理：条件付き確率の計算、ナイーブベイズとの関連
  • 中心極限定理：サンプル平均の分布、統計的推論の基礎
  • 最尤推定：パラメータ推定の原理、対数尤度

よくある間違い

❌ 注意すべきポイント

  • 相関と因果の混同：相関があっても因果関係があるとは限らない
  • 平均と中央値の誤解：外れ値がある場合の代表値選択
  • 確率の加法定理の誤用：排反事象でない場合のP(A∪B)計算
  • 条件付き確率の逆転：P(A|B)とP(B|A)は一般的に異なる
  • 分散の計算間違い：不偏分散と標本分散の使い分け
  • 検定の多重性：複数検定による第一種過誤の増加

まとめ

AIに必要な数理・統計知識は、機械学習アルゴリズムの理解と実装において不可欠な基盤です。確率統計の基本概念から最適化数学まで、幅広い数学的素養が求められます。

特に重要なのは、確率分布の性質、統計的推定・検定、距離と類似度の概念、最適化手法の原理です。これらの知識は、データの特性理解、モデル選択、性能評価において中心的な役割を果たします。

G検定では、各統計量の定義と計算方法、確率分布の特徴、相関と因果の区別といった基本的な統計概念の理解が問われます。実際の機械学習プロジェクトにおいても、これらの知識は問題設定から結果解釈まで一貫して活用されます。

主な用語集

重要用語

確率分布
確率変数がとりうる値とその確率を対応づけた関数。データ生成過程をモデル化。
期待値
確率分布の平均値。確率変数の中心傾向を表す代表値。
分散
確率変数の散らばりの度合いを表す指標。期待値からの偏差の二乗の期待値。
相関係数
2つの変数間の線形関係の強さを-1から1の値で表す指標。
共分散
2つの確率変数の共変動を表す指標。相関係数の分子部分。
最尤法
観測データが最も尤もらしく生じるパラメータを推定する統計的手法。
最小二乗法
誤差の二乗和を最小化することによりパラメータを推定する回帰分析の基本手法。
正規分布
自然界で最も基本的な連続確率分布。中心極限定理により多くの現象で近似的に成立。
ベルヌーイ分布
成功・失敗の2つの結果のみをもつ試行の確率分布。二項分布の基本要素。
ポアソン分布
単位時間あたりの発生回数をモデル化する離散確率分布。稀な事象の統計モデル。
条件付き確率
ある事象Bが起こったという条件のもとで事象Aが起こる確率。P(A|B)で表記。
帰無仮説
統計的検定において、効果や差がないという基準となる仮説。H₀で表記。
対立仮説
帰無仮説に対立する仮説。研究者が証明したい仮説。H₁で表記。
疑似相関
実際には因果関係がないが、第三の変数の影響で見かけ上の相関が現れる現象。
ユークリッド距離
n次元空間での2点間の直線距離。最も基本的な距離尺度。
マハラノビス距離
データの分散共分散構造を考慮した距離尺度。外れ値検出で重要。
コサイン類似度
2つのベクトルの角度の余弦値で表される類似度。文書分析やレコメンドで使用。
相互情報量
2つの確率変数間の依存関係の強さを表す情報理論の指標。
移動平均
時系列データの一定期間の平均値を逐次計算する平滑化手法。
外れ値
データ分布の大部分から大きく外れた値。異常検知や前処理で重要。
